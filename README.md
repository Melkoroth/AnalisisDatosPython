# Análisis de datos con python

## Instalación
* Se requiere Anaconda (Python 3.6 + NumPy + pandas + Sci-Py + Scikit-Learn...)
* Se usa jupyter notebook. Ir a la carpeta del proyecto y ejecutar "jupyter notebook"
